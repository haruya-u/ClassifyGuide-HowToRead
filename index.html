読み方の選択ガイド

本を開く前の30秒で、読み方の精度を上げる

「この本は何をしようとしているか」を最初に問う。それだけで、読み方の精度が大きく変わる。

Question 1 / 3
この本は、データや論理を根拠に何かを主張しているか？
目次・まえがきを見て「仮説を証明しようとしている雰囲気」があるか
Question 2 / 3
その主張は、客観的な方法で反証できるか？
実験・統計・再現性など、第三者が検証できる論拠があるか。「著者の解釈」だけで成り立っていないか
Question 2 / 3
著者の価値観や立場からの評価・判断が含まれているか？
「○○は正しい／間違い」「○○であるべき」という著者の価値判断が前面に出ているか
Question 3 / 3
知識や情報を整理・体系化して伝えることが主な目的か？
著者の主張よりも「知識の地図を渡すこと」が目的に見えるか
TYPE A
証明する本
論理の型：帰納・演繹・数式的な論証
📖 読み方：一文一文の論証を追う精読
著者が「なぜそう言えるか」を論証しようとしている本。論拠の一つひとつを追い、飛躍や矛盾がないかを確かめながら読む。「面白い主張だ」と思っても、それが証明されているかどうかは別問題として扱う。
読みながら問うこと
この論拠は反証できるか？反証例を著者はどう扱っているか？
データの選び方に恣意性はないか？都合の良いデータだけを使っていないか？
前の論点から次の論点への飛躍はないか？
「相関」を「因果」として扱っていないか？
⚠️ 見せかけ型に注意：データや統計を多用しながら、実は著者の世界観を広めようとしているType Cの本が多い（ベストセラー教養書に頻出）。読み進めて「主張が強い割に反証を扱っていない」と感じたら、Type Cとして再診断を。
例：科学書、学術論文、統計分析書
TYPE B
考え抜く本
論理の型：弁証法的・止揚（アウフヘーベン）
📖 読み方：自分の仮説を持ちながら読む精読
著者が「答えを出す」のではなく、「問いを深める」ことを目的にしている本。答えを探しながら読むのではなく、著者の問いを自分の問いとして引き受け、自分なりの立場を持ちながら読む。著者と対話するように読むことが大切。
読みながら問うこと
著者はどんな問いを立てているか？その問いは正しく設定されているか？
自分はこの問いにどう答えるか？著者とどこで意見が分かれるか？
著者が乗り越えようとしている対立構造は何か？
論証が難解な場合、「なぜこの言い方をするのか」という著者の意図を探る
例：哲学書、思想書、倫理学書
TYPE C
評価する本
論理の型：規範的・社会的共感の論理
📖 読み方：著者の立場を先に把握してから精読
著者がある立場・価値観から対象を評価・批判している本。読む前にまず「著者はどのイデオロギー・立場から書いているか」を把握する。その前提ごと検討する姿勢が必要で、著者の立場を知らずに内容だけ読んでも正確には理解できない。
読みながら問うこと
著者の立場・イデオロギーは何か？（経済的合理性か、共同体的価値観か、など）
その立場から見えにくくなっているものは何か？
著者が「良い／悪い」と判断する基準は何か？その基準自体は妥当か？
自分はどの立場にいるか？著者と同じ立場なら共感しやすいが、それは論証の妥当性とは別問題
例：社会批評、文芸批評、思想・政治評論
TYPE D
教える本
論理の型：帰納的・事実の列挙・体系化
📗 読み方：全体構造を掴んでから必要箇所を深掘り
知識や情報を体系的に整理して伝えることが目的の本。最初から全部精読しなくていい。目次を地図として使い、全体の構造をざっくり掴んでから、必要な部分だけ戻って深掘りする。
読みながら分けること
これは「事実・データ」か「著者の解釈・見解」か？
知識の整理なのか、特定の結論に誘導しようとしているのか？
引用・参照元は信頼できるか？一次情報か二次情報か？
⚠️ 最も見分けにくい見せかけ型：「教える本」の形式をとりながら、特定の世界観や価値観を読者に植え付けようとしているType Cの本が多い（ビジネス書・教養書のベストセラーに頻出）。「なんとなく納得した」で終わるなら、著者に誘導されている可能性がある。
例：教科書、入門書、専門書、ハンドブック
TYPE E
語りかける本
論理の型：感情・共感・ナラティブ
📘 読み方：流れを止めずに没入する
著者の経験・物語を通じて読者に何かを感じさせることが目的の本。分析より没入を優先する。途中で「これはこういう意味だ」と立ち止まって考えない。読み終えてから、体験した世界を振り返る。
読み終えてから問うこと
著者の体験から、何が普遍的に言えるか？
自分の経験・感覚とどこが重なり、どこが違ったか？
著者が伝えたかったことの核心は何か？
例：小説、自伝、エッセイ、ノンフィクション
⚠ 見せかけ型に注意
データや統計を多用しながら「証明する本（A）」や「教える本（D）」に見えて、実は特定の価値観・世界観を読者に広めようとしている本がある。ベストセラーの教養書・ビジネス書に特に多い。
読み進めて「反証を扱っていない」「主張が強い」と感じたら、Type Cとして読み直すこと。

このガイドについて

オードリー・タン氏の読書論を起点に、本の性格・論理の型・読み手の立場の3軸から読み方を選ぶフレームワークです。本を読む前に「この本は何をしようとしているか」を問うことで、理解の精度が変わります。

使い方

「診断する」タブで3問に答えると、タイプと読み方が表示されます。読み進めて「思っていたタイプと違う」と感じたら、再診断してください。タイプ分類は固定ではなく、読みながら修正するものです。
