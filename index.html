読み方の選択ガイド

本を開く前の30秒で、読み方の精度を上げる

読む前に、この本が「何をしようとしているか」を30秒で問いましょう。

Question 1 / 3
この本は、データや論理を使って何かを主張しているか？
Question 2 / 3
その主張は検証可能か？
（実験・統計など客観的な方法で確かめられそうか）
Question 2 / 3
この本は、何かについて価値判断を含んでいるか？
（「○○は良い・悪い」「○○であるべき」など著者の評価がある）
Question 3 / 3
知識や事実の整理・説明が中心か？
（「教える」ことが主な目的か）
TYPE A
証明する本
📖 読み方：じっくり精読
論拠の一つひとつを追い、飛躍がないか検証しながら読む。「なぜそう言えるのか」を常に問いかけることが大切。
例：科学書、学術論文、統計分析書
⚠️ データを多用しながら特定の世界観を広めようとしている本も多い。「教える本」に見えても実は「評価する本」のことがある。読み進めて違和感を感じたら分類を見直そう。
TYPE B
考え抜く本
📖 読み方：じっくり精読
著者の問いを自分の問いとして引き受け、自分なりの答えを持ちながら読む。答えを探すのではなく、思考を深めるために読む本。
例：哲学書、思想書
TYPE C
評価する本
📖 読み方：立場を意識して精読
著者がどの立場から書いているかを先に把握し、その前提ごと検討する。「誰がどの立場でこう言っているか」を常に意識する。
例：社会批評、文芸批評
TYPE D
教える本
📗 読み方：流し読み＋必要箇所を参照
全体構造をざっくり掴み、必要な部分に戻って深掘りする。最初から全部精読しなくていい。目次を地図として使おう。
例：教科書、入門書、専門書
⚠️ 「教える本」に見せかけて特定の世界観を広めようとしているベストセラーも多い。読み進めて「主張が強いな」と感じたらType Cとして再分類を。
TYPE E
語りかける本
📘 読み方：流し読み
流れを止めず受け取る。著者の体験世界にまず入り込む読み方。分析より没入を優先しよう。
例：小説、自伝、エッセイ
⚠️ 注意：データを多用しながら「教える本」や「証明する本」に見せかけて、実は特定の世界観を広めようとしている本がある（ベストセラーの教養書・ビジネス書に多い）。読み始めて「思っていたタイプと違う」と感じたら、分類を見直すこと。

例：科学書、学術論文、統計分析書

このガイドについて

オードリー・タン氏の読書論をベースに考案した読み方の分類フレームワークです。本を読む前に「この本は何をしようとしているか」を問うことで、読み方の精度が大きく変わります。

使い方のコツ

「診断する」タブで3つの問いに答えると、その本のタイプと推奨読み方が表示されます。読み進めて「思っていたタイプと違う」と感じたら、遠慮なく再診断してください。
